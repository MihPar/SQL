Оператор SELECT

SELECT name, appearance
FROM superheroes

SELECT name AS hero_name, appearance
FROM superheroes

SELECT DISTINCT(align)
FROM superheroes

Ограниченный список элементов

SELECT DISTINCT(align)
FROM superheroes
LIMIT 10

DISTINCT - выбор уникального элеметна
LIMIT - ограниченное значение извлекаемых строк
AS - создать псевдоним

SELECT *
FROM superheroes
WHERE align="Famile Characters"

Операторы сравнения

SELECT *
FROM superheroes
WHERE year BETWEEN 2000 AND 2005

BETWEEN - значение находиться в указанном диапазоне
IN - значение входит в список
LIKE - проверка строк на соответствие шаблона
% - любое колличество символов
_- ровно один любой символ

Логический оператор в WHERE

SELECT *
FROM superheroes
WHERE gender="Female Characters"
AND
align="Bad Characters"
ORDER BY appearance DESC

AND - "И"
OR - "или"
NOT - "нет"

Сортировка и фильтры в select

SELECT *
FROM superheroes
WHERE align="Bad Characters"
ORDER BY appearance DESC

DESC(descending) - сортировка по убыванию
ASC(ascending) - сортировка по возрастанию

Ограничить колличество строк

SELECT *
FROM superheroes
WHERE align="Bad Characters"
AND gender = "Female Characters"
ORDER BY appearance DESC
LIMIT 5

Сортировка по нескольким столбцам

SELECT *
FROM superheroes
ORDER BY year, appearances
